Unordered List/Bulleted List
A list of things in no particular order (e.g., list of ingredients required in a recipe). Each item in this type of list is usually prefaced with dots (bullets), squares, or other shapes.
Ordered List/Numbered List
A list of things in a specific order (e.g., the steps in a recipe). Each item in this type of list is usually prefaced with numbers (e.g. regular numbers or Roman numerals) in numerical order (e.g., 1. 2. 3.).
Definition List
A list where each item contains a term and its corresponding definition — carrying a "he said, she said" or "call and response" pattern with it.
